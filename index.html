    My Developer Story

    I used the internet for the first time when I was 6 years old ... It was pretty awesome. Then I played a computer game. Then I thought, these are man-made. But how? It was all like a miracle to me. Years later, my dad gave me a personal computer as a gift. When I was 12, I wanted to learn programming. I found a guide to the C ++ programming language and started learning. But I couldn’t create anything other than a calculator program. I had no other guide. And the internet was very expensive for us. At school we were taught HTML. I created HTML websites. Then they taught Pascal programming language. I created small games in this language. But that didn’t seem to be enough for me. Then I started studying physics. I haven’t written any code at all during this time. Then quarantine began due to Covid-19 disease. During this time I had time to think carefully. And I went back to programming again. But the knowledge I learned was old for 2020. I learned the Python programming language. I created a desktop game in Python. I wanted to learn web programming due to the lack of tutorials in Uzbek language about Python. I learned HTML, CSS, and started creating websites. But HTML and CSS were not enough. And I learned the JavaScript programming language.
